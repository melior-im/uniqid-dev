Uniqid Generator

Generate IDs using the uniqid package online in your browser or via an API.

API

Get one Uniqid

GET https://uniqid.dev/api

Get multiple Uniqids

GET https://uniqid.dev/api?count=5

Uniqid with a prefix

GET https://uniqid.dev/api?prefix=user_

Uniqid with a suffix

GET https://uniqid.dev/api?suffix=_demo

Uniqid in time mode

GET https://uniqid.dev/api?mode=time

Uniqid in process mode

GET https://uniqid.dev/api?mode=process

Kitchen sink example

GET https://uniqid.dev/api?count=5&prefix=user_&suffix=_demo&mode=process

Output format: plain text (one per line)

Accept header: text/plain

curl -X GET "https://uniqid.dev/api?count=5" -H "Accept:text/plain"

Output format: CSV

Accept header: text/csv

curl -X GET "https://uniqid.dev/api?count=5" -H "Accept:text/csv"

Output format: JSON

Accept header: application/json

curl -X GET "https://uniqid.dev/api?count=5" -H "Accept:application/json"
